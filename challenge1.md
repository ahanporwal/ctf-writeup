# Challenge: <Double Check>

## Problem
We're planning on deploying some new static sites for our officers. We've cloned a template from Hugo's Static Site Generator (SSG). Can you make sure that our website is clean before it's deployed?

## Solution
The flag was hidden somewhere on the author's github profile. My strategy was to check where recent edits were made and from there, figure out which edit held the flag. I then went through the entire github page and scrubbed through the last visited for every segment. Upon finding nothing, I went to the author's profile for further investigation. I found the last update there was a comment and upon clicking on it, the flag was there in plain sight.
